This Agreement strictly complies with TikTok Developer Policies (2026), TikTok Global Data Processing Agreement (GDPA), COPPA, GDPR, CCPA, and all applicable local laws and regulations, including TikTok Content Posting API Policy.

1. Agreement Declaration and Scope of Application

This Agreement is a legally binding contract between you ("User") and 佛山市赫本服装实业有限公司 ("we", "us", "our") regarding your use of TikTok Shop Analytics Tool ("Application") and related TikTok Developer API services (including TikTok Content Posting API for video publishing).

The Application is developed based on TikTok Developer API to provide services in accordance with TikTok-approved business scenarios. Your use of the Application constitutes acceptance of all terms of this Agreement.

2. User Authorization and API Norms

(1) Authorization Process

To use core functions, you must complete TikTok OAuth 2.0 authorization, explicitly granting permission to access your TikTok account public data. We only apply for the minimum necessary permissions:

  • Basic Permissions: Apply for shop.data.read, video.list, user.info.basic permissions for data analysis; apply for video.upload, video.publish permissions for authorized video publishing (only with your explicit consent)
  • Sensitive Permissions: Never apply for comment.write, message.read, user.phoneNumber.read or other sensitive permissions unrelated to data analysis and video publishing

You may revoke authorization at any time in TikTok Account Settings > Authorized Apps. We will immediately stop using your data and terminate pending video publishing tasks upon revocation.

(2) API Usage Restrictions

We strictly comply with TikTok API rate limits and usage rules, and prohibit:

  • Crawling non-public TikTok data or violating TikTok's robots.txt rules
  • Publishing infringing, illegal, or misleading content on behalf of users (all published videos comply with TikTok Community Guidelines)
  • Reselling or renting TikTok API permissions
  • Exceeding TikTok Content Posting API rate limits (≤6 requests per minute per access_token, ≤50 videos per day per account)
  • Any behavior that harms TikTok's platform ecosystem

We will never exceed authorized API call limits or use APIs for unapproved purposes, including unauthorized video publishing without user consent.

3. Data Collection, Usage and Protection

(1) Data Collection Scope (Data Minimization Principle)

We only collect necessary TikTok public data as required by TikTok policies:

  • Public nickname
  • Avatar URL
  • User openID (only for identity identification)
  • Device IP (only retain network segment, anonymized by SHA-256)
  • TikTok Shop operational data (order volume, unit price, transaction amount, inventory status)
  • Video performance data (play count, like count, comment count, conversion rate)
  • Public competitor shop data (transaction data, best-selling product info, video performance)
  • Public video scripts of best-selling products
  • Voluntarily submitted video files/links, captions, hashtags (only for authorized video publishing, deleted immediately after successful publishing)

WE DO NOT collect sensitive data (phone number, precise GPS, biometrics, user private messages, payment information, trade secrets of competitors).

(2) Specific Business Purposes (TikTok-Approved)

We use the collected TikTok public data ONLY for the following specific, TikTok-approved business purposes:

  1. Analyze TikTok Shop operational data (order volume, unit price, transaction amount, inventory status) and video performance data (play count, like count, comment count, conversion rate) to generate real-time operational reports for merchants, helping them optimize product pricing and content release strategies.
  2. Collect and analyze public competitor data (shop transaction data, best-selling product information, video operation performance) to provide merchants with market trend analysis and operational optimization suggestions (only use publicly disclosed data, no access to trade secrets or private information of competitors).
  3. Extract and analyze video scripts of best-selling products (only from public TikTok videos) to help merchants optimize content creation strategies, improve video conversion rates, and comply with TikTok Content Policy and Intellectual Property Rules.
  4. Authenticate user identity via TikTok Login Kit (apply for basic permissions as specified in Section 2.1).
  5. Ensure service security and prevent fraudulent activities (e.g., abnormal order detection, account anti-counterfeiting) in accordance with TikTok Security Rules and anti-cheating mechanisms, without interfering with TikTok's platform operation.
  6. Publish videos to TikTok via official Content Posting API (only with your explicit authorization) to implement scheduled/batch video publishing strategies, fully compliant with TikTok content and rate limit requirements.

(3) Data Compliance Commitment

  • Data transmission: All data is encrypted via TLS 1.3 (end-to-end encryption) when interacting with TikTok US servers, fully compliant with TikTok Global Data Processing Agreement (GDPA) and Standard Contractual Clauses (SCCs) under GDPR, including video publishing data.
  • Data storage: Stored on Alibaba Cloud servers in China (mainland), with cross-border transmission of necessary data to TikTok US servers approved by Chinese regulatory authorities (in accordance with the Measures for Standard Contracts for Outbound Transfer of Personal Information).
  • Data retention: 90 days for business data (immediately deleted after user account cancellation); 30 days for access logs (permanently deleted after expiration, no backup retention); video files/links for publishing are deleted immediately after successful publishing (no backup).
  • Data usage boundary: No secondary use, no data monetization, no user profiling, and no personalized advertising based on collected data.

(1) User Rights

  • Right to access: Request a copy of all your data we collect
  • Right to correction: Request correction of inaccurate personal data
  • Right to deletion: Request permanent deletion of your data (except legal retention requirements)
  • Right to withdraw consent: Revoke API authorization at any time, including revocation of video publishing permissions

  • 转售、出租TikTok API权限
  • 超出TikTok Content Posting API频率限制（单access_token每分钟≤6次请求，单账号每日发布视频≤50条）

我们绝不会超出授权的API调用频次，也不会将API用于未批准的用途，包括未经用户同意的视频发布行为。

三、数据收集、使用与保护

（一）数据收集范围（数据最小化原则）

我们仅根据TikTok政策要求收集必要的用户数据：

  • 公开昵称
  • 头像URL

  • 设备IP（仅保留网段，SHA-256匿名化处理）
  • TikTok店铺运营数据（订单量、客单价、成交金额、库存状态）
  • 视频表现数据（播放量、点赞数、评论数、转化率）
  • 竞品公开店铺数据（交易数据、热销品信息、视频运营表现）

我们不会收集敏感数据：不获取手机号、精确GPS定位、生物识别信息、用户私信、支付信息、竞品商业秘密等TikTok禁止收集的敏感数据。

（二）特定商业场景（TikTok批准）

我们仅将收集的TikTok公开数据用于TikTok批准的以下特定商业场景：

  1. 分析TikTok店铺运营数据（订单量、客单价、成交金额、库存状态）及视频表现数据（播放量、点赞数、评论数、转化率），为商家生成实时运营报表，助力商家优化商品定价、内容发布策略。
  2. 收集并分析竞品公开数据（店铺交易数据、热销品信息、视频运营表现），为商家提供市场趋势分析及运营优化建议（仅使用公开披露数据，不获取竞品商业秘密或私密信息）。
  3. 提取分析热销品的视频脚本（仅来自TikTok公开视频内容），帮助商家优化内容创作策略、提升视频转化率，且完全符合TikTok内容政策及知识产权规则。
  4. 通过TikTok Login Kit完成用户身份验证（申请本协议第二条第（一）款约定的权限）。
  5. 保障服务安全，防范欺诈行为（如异常订单检测、账号反仿冒），严格遵循TikTok安全规则及反作弊机制，不干扰TikTok平台正常运营。
  6. 在你明确授权下，通过TikTok官方Content Posting API发布视频至TikTok平台，帮助你实现视频定时/批量发布策略，完全符合TikTok内容发布政策及频率限制要求。
